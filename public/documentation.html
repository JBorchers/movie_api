myFlix Documentation

  • Return a list of ALL movies to the user
  • Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title (e.g., “Thriller”)
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info (username)
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister

Description of Endpoints

Business Logic URL HTTP Method Request Body Data Format Response
Display list of ALL movies to user /movies GET None JSON object containing data about all movies
Get movie descriptions /movies/:descriptions GET None JSON object containing specific movie data (description, genres, director,image URL)
Get genre description /movies/:genres GET None JSON object containing data about genres
Get list of directors /directors GET None JSON object containing data about directors (bio, birth year, death year)
Register a user /users POST JSON object holding data about the registering user JSON object holding data about the user that was registered, including a unique ID
Update a user's information /users/:username PUT None A text message indicating the user's information has been updated
Add movie to user's list of favorites /users/:username/favorites POST JSON object holding data about the movies to add to favorite's list JSON object holding data about the movies added to favorie's list
Remove movie from user's favorites list /users/:username/favorites/movies DELETE None A text message indicating a movie has successfully been removed
Deregister user /users/:username DELETE None A text message indicating the user has successfully deregistered
